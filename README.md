# Mini Hollow Knight (Python)

## üá∫üá∏ English

A simple terminal-based combat game inspired by the Nightmare King Grimm fight from *Hollow Knight*.

The player must react correctly to different enemy attacks. Each attack requires a specific response, and mistakes reduce the player's health. The fight continues until either the player or the boss is defeated.

This project was created to practice Python fundamentals such as loops, conditionals, functions, and basic game logic.

---

### How to Run

1. Make sure you have Python 3 installed on your machine.
2. Clone this repository:
https://github.com/eojunoneee/mini-hollow-python.git
3. Enter the project folder:
cd mini-hollow-python
4. Run the game:
game.py

The game runs in the terminal and will prompt you to choose a reaction during the fight.

---

### Features

- Turn-based combat loop  
- Random enemy attacks with weighted probability  
- Correct/incorrect reaction system  
- Win and defeat conditions  
- Clean and organized code using functions  

---

### Technologies

- Python 3  
- Standard library (`random`)  

---

### What I Learned

- How to structure a program using functions  
- How to separate responsibilities in code  
- How to manage game state (health, attacks, loop control)  
- How to improve code readability and maintenance  

---

### Future Improvements

- Add new attack phases
