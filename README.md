# Mini Hollow (Python)

Turn-based combat game inspired by Hollow Knight.

## Description
Mini Hollow is a turn-based combat game inspired by Hollow Knight, developed in Python.  
The project was created to practice programming logic, control flow, and basic game mechanics such as loops, conditionals, and random behavior.

This project is part of my learning journey and portfolio as a beginner developer.
